# Чебур-Браузер / Chebur-Browser / Чэбур-Браўзер

Десктопный браузер для российского интернета на базе Chromium (Electron).

Desktop browser for the Russian internet built on Chromium (Electron).

Дэсктопны браўзер для расійскага інтэрнэту на базе Chromium (Electron).

---

## Русский

### Возможности

- Вкладки с навигацией, назад/вперёд, F11 fullscreen
- Контекстное меню (ПКМ): открыть/скопировать/скачать картинку, копировать ссылки
- Реальная погода (wttr.in) с выбором 15 городов
- Курс валют ЦБ РФ в реальном времени
- Часы в тулбаре
- Настройки: язык (Русский, Беларуский), поисковик (Яндекс, Mail.ru, Рамблер), DNS
- DNS: МТС, Яндекс, ТСПУ
- GOST-криптография (ГОСТ Р 34.10-2012, ГОСТ Р 34.11-2012)
- Сертификаты НУЦ Минцифры
- Блокировщик рекламы и трекеров

## English

### Features

- Tabs with navigation, back/forward, F11 fullscreen
- Context menu (right-click): open/copy/save image, copy links
- Live weather (wttr.in) with 15 Russian cities
- Real-time CBR exchange rates
- Clock in toolbar
- Settings: language (Russian, Belarusian), search engine (Yandex, Mail.ru, Rambler), DNS
- DNS: MTS, Yandex, TSPU
- GOST cryptography (GOST R 34.10-2012, GOST R 34.11-2012)
- NUC Ministry of Digital Development certificates
- Ad and tracker blocker
